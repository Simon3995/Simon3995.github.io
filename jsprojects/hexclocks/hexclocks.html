Inspired by the original hex clock

True Hex Clock

The current time (HH:MM:SS) is directly converted into a hex color code
#000000

Scaled Hex Clock

The red value counts up to FF (255) in 24 hours,
The green value counts up to FF (255) in 1 hour,
The blue value counts up to FF (255) in 1 minute
#000000

Full Hex Clock

Counts from 000000 (0) to FFFFFF (~16.8 million) in 24 hours
#000000
